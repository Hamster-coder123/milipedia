Military Aircraft Encyclopedia Plan

Milipedia

A project plan for a Wikipedia-style reference website covering military aircraft across countries, eras, roles, operators, prototypes, retired aircraft, and cancelled programs.

Open the Plan View on GitHub

Planned Scope

Aircraft Database

Structured encyclopedia entries for fighters, bombers, transports, helicopters, UAVs, trainers, and more.

Search and Filters

Find aircraft by name, country, manufacturer, role, type, era, status, NATO reporting name, and operator.

Comparison Tool

Compare 2 to 4 aircraft side by side across specifications, service history, armament, and status.

Source-Driven Data

Use consistent units, neutral language, citations, and clear labels for estimates or disputed claims.

This page is a project-plan preview only. The full Milipedia website has not been implemented yet.
